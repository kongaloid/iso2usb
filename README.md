iso2usb is a Python script to help you make a bootable USB drive from a Linux ISO.

iso2usb runs on MacOS.

Run it from terminal, like so:

    python iso2usb.py [path to USB] [path to ISO]

===================================================================

iso2usb.py -h
	
	positional arguments:
	  usb         Full path to USB, e.g.: /dev/disk2
	  iso         Full path to ISO e.g.: /Users/username/Desktop/ubuntu-amd64.iso
	
	optional arguments:
	  -h, --help  show this help message and exit
